Responsive Video Gallery WordPress plugin

== Description ==

Responsive Video Gallery is a very effective WordPress video gallery plugin which allows you to make nice looking video galleries with responsive layouts and customizable styles. It's very easy to use & manage everything via options page. Need some simple step to create your video gallery.

== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "Wp Video Gallery" menu at left side on WordPress dashboard and find "Wp Video Gallery" Settings.<br />
<br />

<strong>Using FTP</strong>
1.Extract wp-responsive-video-gallery.zip to your computer.
2.Upload the wp-responsive-video-gallery directory to your wp-content/plugins directory.
3.After activate plugin you will see "Wp Video Gallery" menu at left side on WordPress dashboard and find "Wp Video Gallery Settings"

<br />
<strong>How to use short-code?</strong><br />
When Wp Video Gallery setup done please copy & paste shortcode into your theme any where you want to display this Video Gallery.
